#### 获取数据列表

**请求格式**

```
我的作品列表
/works?title=***&pageIndex=0&pageSize=4
公共模版列表
/templates?title=***&pageIndex=0&pageSize=4
```

**基望的返回数据格式：**

```
{
  "count": 总数,
  "pageSize": 5,
  "pageIndex": 0,
  "list": [
    {
      "id": 2,
      "author": "",
      "copiedCount": 0,
      "coverImg": "",
      "title": "",
      "isHot": "",
      "createdAt": "",
      "user": {
        "username": "",
        "nickName": "",
        "picture": ""
      }
    },
    ...
  ],
}
```

### 查询列表的可变条件

* 页数
* 每页多少条记录返回哪些值 - projection / select
* 关联哪些集合以及返回哪些值 - $lookup/populate
* 排序 - sort = { key: value }
* 查询条件 - query = { key: value }
